Como usuario, 
quiero iniciar sesión
para acceder a mi cuenta.

Criterios de Aceptación

El usuario deberá poder iniciar sesión utilizando correo electrónico y contraseña.

El sistema deberá validar las credenciales ingresadas.

Si las credenciales son incorrectas, deberá mostrarse un mensaje de error.

Al iniciar sesión correctamente, el usuario deberá acceder a su área correspondiente.

El sistema deberá mantener la sesión activa mientras no cierre sesión.
