Criterios de Aceptación

El usuario deberá poder cerrar sesión desde cualquier página autorizada.

El sistema deberá finalizar la sesión activa.

El usuario deberá ser redirigido a una pantalla pública.

No deberá poder acceder a áreas protegidas sin volver a autenticarse.
