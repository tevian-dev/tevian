Como visitante,
quiero registrarme en la plataforma
para poder acceder a los eventos y funcionalidades.



Criterios de Aceptación

El sistema deberá permitir registrar una nueva cuenta utilizando nombre, correo electrónico y contraseña.

Todos los campos obligatorios deberán estar completos para finalizar el registro.

El correo electrónico no podrá estar registrado previamente.

La contraseña deberá cumplir con las políticas de seguridad definidas por el sistema.

Al finalizar el registro exitosamente, el usuario deberá recibir una confirmación.

La información deberá almacenarse correctamente en la base de datos.
