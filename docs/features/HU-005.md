Como usuario,
quiero visualizar eventos disponibles
para descubrir actividades de interés.

Criterios de Aceptación

El sistema deberá mostrar la lista de eventos disponibles.

Cada evento deberá mostrar título, descripción resumida, fecha, categoría, cupos e imagen.

Los eventos deberán visualizarse tanto para usuarios registrados como invitados.

La información mostrada deberá encontrarse actualizada.
